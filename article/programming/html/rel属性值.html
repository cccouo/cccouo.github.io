    • alternate

    描述：表示当前文档的替代版本，如打印版、翻译版等。

    示例：

                    
                    <link rel="alternate" href="print.html" media="print">
                    <link rel="alternate" hreflang="fr" href="article-fr.html">

    解释：第一个示例指定了当前页面的打印版本；第二个示例指定了当前文章的法语版本。

    alternate

    描述：表示当前文档的替代版本，如打印版、翻译版等。

    示例：

                    
                    <link rel="alternate" href="print.html" media="print">
                    <link rel="alternate" hreflang="fr" href="article-fr.html">

    解释：第一个示例指定了当前页面的打印版本；第二个示例指定了当前文章的法语版本。

    icon

    描述：用于指定网站的图标（favicon），会显示在浏览器的标签页或书签栏上。

    示例：

                    
                    <link rel="icon" href="favicon.ico">

    prev 和 next

    描述：prev 表示当前文档的前一个文档，next 表示当前文档的后一个文档，常用于分页导航。

    示例：

                    
                    <link rel="prev" href="page1.html">
                    <link rel="next" href="page3.html">

    解释：假设当前页面是 page2.html，上述代码分别指定了前一页为 page1.html，后一页为 page3.html。

    up

    描述：表示当前文档的父文档，常用于导航到上一级页面。

    示例：

                    
                    <link rel="up" href="index.html">

    nofollow

    描述：告诉搜索引擎不要追踪这个链接，通常用于用户生成内容中的链接，防止垃圾链接影响网站的搜索引擎排名。

    示例：

                    
                    <a href="https://example.com" rel="nofollow">External Link

    noopener 和 noreferrer

    描述：noopener 用于防止通过 target="_blank" 打开的新窗口访问原窗口的 window.opener 对象，增强安全性；noreferrer 则进一步阻止在链接请求中发送 Referer 头信息。

    示例：

                    
                    <a href="https://example.com" target="_blank" rel="noopener noreferrer">External Link

    解释：这是一段 HTML 代码，表示一个超链接。其中，“<a href="https://example.com" target="_blank" rel="noopener noreferrer">” 定义了一个超链接，“href” 属性指定了链接的目标地址为 “https://example.com”，“target="_blank"” 表示在新窗口中打开链接，“rel="noopener noreferrer"” 用于防止恶意网站通过链接进行攻击。“External Link” 是超链接显示的文本内容。最后以 “” 结束超链接的定义。

    manifest

    描述：用于指定 Web 应用程序的清单文件，该文件定义了应用在离线时的行为。

    示例：

                    
                    <link rel="manifest" href="app.webmanifest">

    author

    描述：指向文档作者的页面。

    示例：

                    
                    <link rel="author" href="author.html">

    help

    描述：指向帮助文档。

    示例：

                    
                    <link rel="help" href="help.html">

    注：如果代码中包含特殊字符（如 < ），在 HTML 中需要进行实体编码，（用& l t 代替<）;否则浏览器可能会将其解析为 HTML 标签，导致 Prism.js 无法正确处理代码。
